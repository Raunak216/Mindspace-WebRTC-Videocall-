# 🧠 MindSpace  
### Real-Time Communication Platform

---

## 🚀 Overview

MindSpace is a real-time communication platform that enables users to connect through **video calls, chat, and screen sharing** in a seamless and interactive way.

It supports secure authentication, instant messaging, and meeting-based communication with options for both authenticated users and guest sessions.

---

## ✨ Features

### 🔐 Authentication
- Secure user authentication using **bcrypt**
- Login and signup functionality
- Password hashing for enhanced security

---

### 📞 Real-Time Communication
- One-to-one or group video communication
- Built using WebRTC for low-latency interaction

---

### 💬 Real-Time Chat
- Instant messaging within sessions
- Messages update in real-time using sockets

---

### 🖥️ Screen Sharing
- Users can share their screen during sessions
- Useful for collaboration and discussions

---

### 🔗 Guest Session Support
- Generate and share meeting links
- Guests can join without creating an account

---

## 🛠️ Tech Stack
- React  
- Node.js  
- Express
- WebRTC  
- Socket.io  
- bcrypt  

---

## ⚙️ How It Works

1. Users sign up or log in securely  
2. Create or join a meeting room  
3. Share the meeting link with others  
4. Communicate via:
   - Video call  
   - Chat  
   - Screen sharing
